# Nativ_Maps
Android library for creating and editing gps routes

Add it in your root build.gradle at the end of repositories:

	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}

Add the dependency

	dependencies {
	        implementation 'com.github.NateHolland:Nativ_Maps:Tag'
	}
